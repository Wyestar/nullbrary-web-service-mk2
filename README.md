# nullbrary web service

# summary
This is the web server and client app for nullbrary, a library/bestiary for information from Capcom's Monster Hunter series of games.

I am building this to practice all aspects of fullstack web development. I do not intend to 'ship' this or gain monetarily from it.

# stack
Written in and built with:
typescript
CSS (tailwind)
react
tanstack router/query/form

Database is in ravenDB with a java service layer for HTTP RESTful api interactions. Controlled request and response structures between the service and this server.
